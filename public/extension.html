Installation

1

Download the Extension

Click the button below to download the Chrome extension package (.zip).

📥 Download Extension
2

Install in Chrome

  1. Open chrome://extensions/ in your browser
  2. Enable Developer mode (toggle in top right)
  3. Drag the downloaded .zip file onto the extensions page
  4. Click Add extension
3

Configure Your Server URL

If your auto-responder is hosted on Railway or another server:

  • Click the extension icon in Chrome's toolbar
  • Double-click the status message to open settings
  • Enter your server's full URL (e.g., https://your-app.up.railway.app)
  • If running locally, the default http://localhost:3000 is pre-set.
4

Sync Your Listings

  1. Log into your marketplace account (Facebook, eBay, Etsy, etc.)
  2. Navigate to "Your Listings" or "Active Listings" page
  3. Click the extension icon → select platform → click "Sync My Listings"
  4. Wait for confirmation. Your listings are now in your auto-responder!

Troubleshooting

No listings found?

Make sure you are on the "Your Listings" page for that platform (not a search page). The extension only scans the current page for listing elements.

Extension won't install?

Chrome requires Developer mode to be enabled for unpacked extensions. Make sure you toggled it on in chrome://extensions/.

Cannot connect to server?

The extension tries to connect to the same domain as this web app by default. If your server is on Railway, double-click the status bar in the extension popup to enter the Railway URL.

Login required?

You must be logged into the marketplace website in your browser before using the sync. The extension uses your existing session; it does not store or transmit your credentials.
